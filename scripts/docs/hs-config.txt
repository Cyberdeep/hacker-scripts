DESCRIPTION
-----------

hs-config

This command creates a new configuration file named 'config.ini' at 
the location where hacker-scripts is installed.

If it already exists you will be asked if you want to overwrite the
file, if you cancel the overwrite operation the configuration file 
will open in notepad for you to edit.

USAGE
-----

hs-config [-h]

Required aruements:

None

Optional arguements:

--h, -h		Display this help documentation
