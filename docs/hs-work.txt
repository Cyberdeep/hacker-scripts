DESCRIPTION
-----------

hs-work

This command will open all the files specified in the configuration with
a text editor, which again will be specified in the configuration.

CONFIGURATION
-------------

Here's an example configuration:

[hs-work]
	editor = C:\Program Files\Sublime Text 3\sublime_text.exe
	project1 = C:\Users\Areeb\Desktop\Python\My Stuff\Projects\Python 3\RSS2IRC

1. EDITOR

The editor/program to open the project files with.

2. PROJECTS

The projects you want to open with the text editor.

USAGE
-----

hs-work [-h]

Required arguments:

None

Optional arguments:

--h, -h		Display this help documentation
